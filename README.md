# payment-reminder-automation-n8n
Automated payment reminder workflow using n8n and Excel to send email alerts on due dates.


This is a simple automation built using **n8n** to send payment reminder emails automatically.

## What this automation does
- Runs daily using a Schedule Trigger
- Reads payment data from Google Sheets
- Checks if the due date matches today
- Sends a reminder email automatically using Gmail

## Why I built this
I wanted to avoid manual follow-ups and learn how real-world automation works using n8n and JavaScript.

## Tools Used
- n8n
- Google Sheets
- JavaScript
- Gmail

## Use Case
This automation can be used for:
- Payment reminders
- Invoice follow-ups
- Task due date alerts
